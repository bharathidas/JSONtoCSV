# JSON to CSV

## Features
### •	data
array of objects.
### •	filename 
The filename. The .csv extension will be added if not included in file name. 
### •	delimiter
fields separator.
### •	headers
List of columns that will be used in the final CSV file. provide in comma separated string.

## Dependencies:

###  Mendix modeler 9.24.18.
